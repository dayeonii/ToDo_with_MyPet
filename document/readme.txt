<실행 방법>

- 권장 최소 사양 : Android 5.0 (Lollipop), API 레벨 21
- 안드로이드 스튜디오의 에뮬레이터를 사용할 경우 구글 플레이스토어가 지원되는 장치 사용
- 실행 안정성을 위해 최신 버전 권장

1. 안드로이드 스튜디오 에뮬레이터로 실행

    1) 프로젝트 파일인 project_file 을 다운로드하여 안드로이드 스튜디오에서 프로젝트를 불러옵니다.
    2) 안드로이드 스튜디오의 Device Manager를 통해 실행할 에뮬레이터를 생성합니다.
    3) 우측 상단의 장치 연결버튼에 'Open Android Emulator: ~~' 를 눌러 장치를 연결합니다.
    4) main.dart 클래스를 실행합니다. (Run 'main.dart')
    5) 어플리케이션이 실행되는 모습을 확인합니다.

2. 빌드된 프로젝트 파일(.apk)을 안드로이드 기기에 설치하여 실행

    1) 첨부된 '파일명.apk'를 안드로이드 모바일 기기에 설치합니다.
    2) 출처를 알 수 없는 앱 설치가 비활성화 되어있는 경우
        a. 설정-보안 및 개인정보 보호-기타 보안 설정-출처를 알 수 없는 앱 설치 로 이동합니다.
        b. 출처를 알 수 없는 앱 설치 버튼을 활성화 합니다.
        c. 실행 파일을 다운로드 받아 실행합니다.
    3) 설치된 어플리케이션을 실행하면 시작 화면이 나타납니다.

2. 어플리케이션 실행 후 사용법

    1) 어플리케이션 실행 시 로그인 화면에서 시작합니다.
    2) Register 버튼을 누르면 회원가입을 할 수 있습니다. 로그인에 사용할 이메일과 비밀번호를 입력 후 등록합니다.
    3) 회원가입에 성공하면 다시 로그인 화면으로 돌아갑니다. 회원가입 시 입력한 정보로 로그인을 합니다.
    4) 로그인에 성공하면 메인 화면인 To-Do List가 나옵니다. 여기서는 + 버튼을 통해 업무를 추가하고, -버튼을 통해 제거할 수 있습니다.
    5) 업무를 일정량 이상 완료하면 아이템 받기 기능을 사용할 수 있습니다.
    6) 펫 화면으로 이동하면 보유중인 아이템으로 펫과 상호작용을 할 수 있습니다.
    7) 친구 화면으로 이동하면 현재 친구 목록을 확인할 수 있습니다. 검색 기능을 지원합니다.
    8) 친구 요청 화면으로 이동하면 현재 내가 받은 친구 요청을 확인할 수 있습니다. 수락/거절을 할 수 있습니다.
    9) 로그아웃 버튼을 누르면 로그아웃 기능이 실행되어 초기 로그인 화면으로 이동합니다.
